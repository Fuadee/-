# Procurement MVP (UI Scaffold)

โครงเริ่มต้นสำหรับ MVP ระบบจัดซื้อจัดจ้างตาม PRD โดยทำเฉพาะหน้า UI ยังไม่ต่อฐานข้อมูล

## Run
```bash
npm install
npm run dev
```

## หน้าจอที่มี
- `/cases` : Case List
- `/cases/new` : Create Case
- `/cases/:id/edit` : Edit Case
- `/cases/:id/preview` : Preview / Generate
